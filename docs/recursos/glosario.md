# Glosario

- **AP** Plan de acción (action plan)
- **CIA** Confidencialidad, integridad y disponibilidad (availability)
- **iSMS** information Security Management System
- **IEC**  International Electrotechnical Commission
- **ISO** International Organization for Standardization
- **RTP** Plan de tratamiento de riesgos (Risk Treatment Plan) 
- **SoA** Declaración de aplicabilidad (Statement of Applicability)
